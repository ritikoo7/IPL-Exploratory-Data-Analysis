# IPL Exploratory Data Analysis 🏏📊

## About the Project

This project presents an **Exploratory Data Analysis (EDA)** of the Indian Premier League (IPL) dataset using Python. The objective is to analyze match data, identify trends, and visualize key statistics related to teams, players, toss decisions, and match outcomes.

The project was created as part of my learning journey in **Data Science and Data Analytics**, helping me gain practical experience with real-world datasets.

## Repository

🔗 **GitHub Repository:**

## Objectives

* Explore the IPL dataset.
* Perform data cleaning and preprocessing.
* Analyze team and player performance.

## Installation

Clone the repository:

```bash
git clone https://github.com/ritikoo7/IPL-Exploratory-Data-Analysis.git
```

Move into the project directory:

```bash
cd IPL-Exploratory-Data-Analysis
```

Install the required libraries:

```bash
pip install pandas numpy matplotlib seaborn
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```
